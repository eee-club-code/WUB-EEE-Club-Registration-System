# WUB EEE Club Registration System

A complete club registration system for the World University of Bangladesh - Electrical and Electronic Engineering Club.

## Features

- 📝 Complete registration form with validation
- 🆔 Auto-generated Club IDs (WUB-EEE-2026-0001)
- 💳 bKash payment integration (200 TK) with TrxID verification
- 📧 Automated email confirmation (simulated)
- 📊 Admin dashboard to view all registered members
- 📄 Downloadable registration slip
- 💾 LocalStorage data persistence
- 📱 Fully responsive design

## Live Demo

[View Live](https://your-username.github.io/WUB-EEE-Club-Registration-System/)

## How to Use

1. Fill out the registration form
2. Pay 200 TK via bKash (01754526593)
3. Enter your bKash TrxID and verify
4. Complete registration
5. Receive confirmation email
6. Download your registration slip

## Technology

- HTML5
- CSS3 (with animations)
- Vanilla JavaScript
- Font Awesome icons
